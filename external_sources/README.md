# Arquivos externos baixados

Este diretório é destinado a conter os repositórios clonados externamente (external_sources/).

A rotina sugerida é:
1. Rodar scripts/clone_external.sh localmente ou em runner com permissão de internet.
2. Inspecionar cada repositório em external_sources/ e extrair README.md, docs e examples conforme sources.yaml.
3. Gerar os documentos padronizados dentro de MASTER_BRAIN/ correspondentes a cada fonte.
